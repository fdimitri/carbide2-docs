## How to use this page

- If you are changing the collaboration model, start with the center column and the OT/CRDT role.
- If you are changing deployment or workspace runtime boundaries, start with the Kubernetes role.
- If you are changing interaction design or contributor experience, start with the UX and docs roles.
- If you want to help fill one of the open roles, reach out before pushing a large design change.
