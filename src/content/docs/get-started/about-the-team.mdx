## Team

### Frank

<img src="/team/frankd.jpg" alt="Frank" width="320" style="border-radius: 12px; border: 1px solid #1c2230;" />

**Role:** Founder and Principal Engineer

**Contact:** FrankD@carbidecore.online

Frank is building CARB/IDE2 as an integrated collaborative development runtime:
editor, terminals, chat, video, and agents in one workspace-first system.

Focus:
- Shipping a practical server-authoritative collaboration model now
- Keeping terminal execution isolated via the PTY sidecar boundary
- Tightening realtime contracts, fanout semantics, and reliability behavior
- Preserving a path to OT/CRDT later without throwing away durable history
