## Authentication

### `POST /api/auth/login`

Issue a JWT for an existing user.

**Request body:**
```json
{ "email": "user@example.com", "password": "..." }
```

**Response:**
```json
{ "token": "<jwt>", "user": { "id": 1, "email": "..." } }
```

### `POST /api/auth/signup`

Register a new user and receive a JWT.

## Concrete examples

### Example: list project tree

```bash
curl -H "Authorization: Bearer $TOKEN" \
  "http://localhost:3000/api/projects/1/fs/tree?path=/"
```

Example response:

```json
{
  "entries": [
    { "name": "src", "ftype": "folder", "size": null, "mtime": "2026-07-14T18:10:22Z" },
    { "name": "README.md", "ftype": "file", "size": 1387, "mtime": "2026-07-14T18:10:32Z" }
  ]
}
```

### Example: create a file

```bash
curl -X POST \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"path":"/src/new_file.rb","content":"puts :ok\n"}' \
  "http://localhost:3000/api/projects/1/fs/files"
```

## Projects

### `GET /api/projects`

List all projects the authenticated user is a member of.

### `POST /api/projects`

Create a new project.

**Request body:**
```json
{ "project": { "name": "my-project" } }
```

### `GET /api/projects/:id`

Get a single project.

### `PATCH /api/projects/:id`

Update project metadata.

### `DELETE /api/projects/:id`

Delete a project. Owner only.

## Filesystem (VFS)

All VFS endpoints are scoped to a project: `/api/projects/:project_id/fs/...`

<Aside type="tip">
  The VFS is also accessible over WebSocket via the `fs` commandSet.
  See [WebSocket Commands](/api-reference/websocket-commands/).
</Aside>

### `GET /api/projects/:project_id/fs/tree`

List directory entries for a given path.

**Query params:** `path` (default: `/`)

### `GET /api/projects/:project_id/fs/files/*path`

Read a file's current content (replays `FileChange` log via `calc_current`).

### `POST /api/projects/:project_id/fs/files`

Create a new file.

**Request body:**
```json
{ "path": "/src/app.js", "content": "" }
```

### `PUT /api/projects/:project_id/fs/files/*path`

Set full file contents.

### `DELETE /api/projects/:project_id/fs/files/*path`

Delete a file or directory.

### `POST /api/projects/:project_id/fs/upload`

Upload an archive (zip, tar, tar.gz) or single file and import into the VFS.

**Content-Type:** `multipart/form-data`
