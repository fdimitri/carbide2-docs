## carbide2-server / worker

| Variable | Default | Purpose |
|----------|---------|---------|
| `DATABASE_URL` | — | Postgres connection (shared by Rails + worker) |
| `SECRET_KEY_BASE` | — | JWT signing + Rails secrets |
| `CARBIDE_CORS_ORIGINS` | local dev hosts | Comma-separated allowed origins; literal or `/regex/` |
| `VFS_FLUSH_INTERVAL_MS` | `800` | VfsFlusher periodic sweep interval |
| `VFS_FLUSH_THRESHOLD_BYTES` | `20` | Immediate flush byte threshold |
| `CARBIDE2_CLIENT` | `../carbide2-client` | Path to the client checkout (dev) |

## carbide2-control

| Variable | Default | Purpose |
|----------|---------|---------|
| `DATABASE_URL` | — | Control-plane Postgres |
| `SECRET_KEY_BASE` | — | JWT signing |
| `KUBECONFIG` | in-cluster | k8s API access for the operator |

> `config/environments/*.rb` and chart `values.yaml` when adding entries.
